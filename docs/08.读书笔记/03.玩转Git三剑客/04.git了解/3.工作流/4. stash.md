### 查看 git stash

列出所有存储

```
git stash list 
```







#### 存储贮藏

```
git stash 
```

如果要加贮藏信息

```
git stash save "saveMessage"
```

如果有新增、删除文件等操作需要贮藏

```
git stash -u
```





### 应用贮藏

##### apply：应用后不销毁

```
git stash apply
```

##### pop：应用后销毁

```
git stash pop
```



### 删除贮藏

```
git stash drop stashName
```
