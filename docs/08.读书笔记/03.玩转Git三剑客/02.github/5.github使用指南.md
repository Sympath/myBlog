### 基础搜索

在不加关键词时，默认只会用搜索词对项目名和项目描述进行搜索

### 高级搜索方案

在我们进行搜索项目时，可以加一些关键词从而进行高级搜索



##### 指定readme文件中对关键词进行搜索：Inreadme

```
git 最好 学习 资料 inreadme
```



##### 指定stars个数

```
git 最好 学习 资料 inreadme stars:>1000
```









### 备注

更多帮助信息可以看github上

help 然后在页面上搜索

```
searching for information on GitHub
```
