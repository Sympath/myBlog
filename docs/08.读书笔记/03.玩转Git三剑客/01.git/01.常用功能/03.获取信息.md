获取npm源



##### git 查看分支创建时间

```
git reflog show --date=iso <branch name>
```

可以查看到指定分支的历次更改记录，最下面一条的时间即是分支创建时间。



##### 查看某一次提交（commit id）中的修改记录

```
git show <SHA-1>
```
